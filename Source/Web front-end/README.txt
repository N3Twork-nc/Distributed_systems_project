1. Start Project:
    - npm install
    - npm start (nodemon index.js) ->  cầu hình "start":"nodemon index.js" in package.json file

2. Lưu ý import file (xem ở index.html)
    - những file nằm ớ thư mục public khi import k sử dụng ("public/css/style.css) mà sử dụng ("css/style.css")
    - có thể truy cập qua url: http://localhost:3000/img/card.jpg mấy file khác cũng làm đc
